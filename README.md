# parisjug-2026

This is the code repository to generate the slides for my talk at Paris JUG on March 10 2026. I share this to illustrate the fact that GitHub Copilot CLI and a few skills are great tools to generate a whole presentation from a simple prompt listing articles, blog posts and papers.

In order to generate the slides, launch GitHub Copilot CLI and type the following prompt: `Generate a presentation based on @prompt.txt`.

You need to create a .env file defining `OPENAI_API_KEY` to use the image generation skill.
I check in the latest js file generated by Copilot to generate the deck, and the last deck version.
This prompt uses my personal preferences for presentations in AGENTS.md, a sub agent to enhance the title slide image prompt ,a skill to generate the title slide image using an image model and a skill to create the slides.
It also uses the workiq plugion to read my email and include some of their content in the deck.
Last, this prompt is multi lingual, mixing french and english, and the model finds it's way through all this.

Once the presentation has been generated you can tweak it by asking the agent to fix things such as image size, layouts, ...
